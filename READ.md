# US Natural gas Data Analysis & Forecasting

This Project performs "data analysis", "visualization", and "machine-learning forecasting" on U.S. Natural gas export data.

It uses 4 datasets that include export olumes,LNG prices, Henry Hub spot prices, and import behaviour from major countries(japan, India, UK, Spain, South Korea).

The goal of this project is to help beginner learn:
  - How to organize a data science project
  - load and merge datasets using python
  - Build machine learning models
  - Make predictions
  - Upload a complete data project to GitHub
